# Marketing Team Help

## Overview

Display the complete marketing team roster, available workflows, and utility commands to help the user navigate the BMAD Marketing system.

## On Activation

Present the following information clearly:

---

### Marketing Team Agents

| Agent | Phase | Command | What They Do |
|-------|-------|---------|-------------|
| Maya | 1 - Research | `/maya` | Market research, audience personas, competitive intelligence |
| Sam | 2 - Strategy | `/sam` | Marketing strategy, positioning, messaging, content calendars |
| Claire | 3 - Creative | `/claire` | Campaign briefs, brand guidelines, copywriting, design briefs |
| Derek | 4 - Execution | `/derek` | Campaign launches, email sequences, ad setup, deployment |
| Aria | 5 - Analytics | `/aria` | Performance reports, A/B test analysis, ROI, pivot recommendations |

### Guided Workflows

| Workflow | Agent | Command | Steps |
|----------|-------|---------|-------|
| Market Research | Maya | `/market-research` | 4 steps: Scope → Audience → Competitive → Synthesis |
| Marketing Strategy | Sam | `/marketing-strategy` | 4 steps: Situation → Positioning → Channels → Campaign Plan |
| Content Calendar | Sam | `/content-calendar` | 3 steps: Pillars → Channel Mapping → Calendar Build |
| Campaign Brief | Claire | `/campaign-brief` | 3 steps: Intake → Creative Direction → Deliverable Specs |
| Brand Guidelines | Claire | `/brand-guidelines` | 3 steps: Audit → Voice & Tone → Visual Identity |
| Launch Campaign | Derek | `/launch-campaign` | 4 steps: Checklist → Setup → Deploy → Go-Live |
| Performance Report | Aria | `/performance-report` | 4 steps: Data → Analysis → Insights → Recommendations |

### Utilities

| Command | Description |
|---------|-------------|
| `/help` | Show this help screen |
| `/marketing-init` | Set up project context (product, audience, goals) |
| `/brainstorm` | Multi-persona brainstorming session |

### Recommended Flow

1. Start with `/marketing-init` to set up your project context
2. Use `/maya` or `/market-research` to research your market
3. Move to `/sam` or `/marketing-strategy` to build your strategy
4. Use `/claire` for creative briefs and brand guidelines
5. Launch with `/derek` when ready to execute
6. Measure results with `/aria`

All generated deliverables are saved to `docs/output/`.
