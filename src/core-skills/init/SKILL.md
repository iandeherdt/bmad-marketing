# Project Initialization

## Overview

Set up the foundational project context that all marketing agents will reference. This creates `docs/project-context.md` with key information about the product, audience, and goals.

## On Activation

### Step 1: Check for Existing Context

Check if `docs/project-context.md` already exists:
- If YES: Display the current context and ask if the user wants to update it or start fresh
- If NO: Proceed to Step 2

### Step 2: Gather Information

Ask the user the following questions one at a time. Be conversational and help them think through each answer:

1. **Product/Service Name**: "What product or service are we marketing?"
2. **Product Description**: "Give me a 2-3 sentence description. What does it do and why does it matter?"
3. **Target Audience**: "Who is your primary target audience? Be as specific as you can — role, industry, company size, pain points."
4. **Secondary Audiences**: "Are there any secondary audiences we should keep in mind? (optional)"
5. **Business Goals**: "What are your top 2-3 marketing goals right now? (e.g., brand awareness, lead generation, user acquisition, retention)"
6. **Current State**: "What marketing are you doing today, if anything? What's working and what's not?"
7. **Competitors**: "Who are your top 2-3 competitors?"
8. **Differentiators**: "What makes you different from the competition? What's your unique value?"
9. **Budget & Resources**: "What's your approximate marketing budget and team size?"
10. **Timeline**: "Any key dates or deadlines we should know about? (launches, events, quarters)"

### Step 3: Generate Project Context

Create `docs/project-context.md` using this structure:

```markdown
# Project Context

## Product
- **Name**: [answer]
- **Description**: [answer]
- **Differentiators**: [answer]

## Audience
- **Primary**: [answer]
- **Secondary**: [answer]

## Goals
[numbered list of goals]

## Current State
[summary of current marketing activities]

## Competitive Landscape
[list of competitors with brief notes]

## Constraints
- **Budget**: [answer]
- **Team**: [answer]
- **Timeline**: [answer]

---
*Generated by BMAD Marketing Init on [date]*
*Update this file anytime by running `/init`*
```

### Step 4: Confirm and Next Steps

After saving, confirm the file was created and suggest:
- "Your project context is ready. All agents will reference this when activated."
- "Suggested next step: `/maya` to start market research, or `/marketing-strategy` if you're ready to plan."
