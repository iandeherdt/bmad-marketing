# Step 1: Data Collection

**Progress: Step 1 of 4** — Next: Analysis

## STEP GOAL

Gather all relevant performance data — metrics by channel, campaign, and time period — and organize it for analysis.

## MANDATORY RULES

- Collect data systematically — don't cherry-pick metrics
- Document data sources so the report is reproducible
- Include both leading and lagging indicators
- Note any data gaps or reliability concerns

## EXECUTION PROTOCOL

1. Define the reporting scope:
   - "What time period are we analyzing? (last week, last month, last quarter)"
   - "Which campaigns or channels are in scope?"
   - "What are we comparing against? (previous period, targets, industry benchmarks)"
2. Collect metrics by channel. Ask the user for each:

   **Website / Landing Pages:**
   - Sessions, unique visitors, bounce rate
   - Page views, time on page
   - Conversion rate, form submissions
   - Traffic sources breakdown

   **Email:**
   - Sends, deliverability rate
   - Open rate, click rate, click-to-open rate
   - Unsubscribe rate
   - Conversions from email

   **Social Media:**
   - Followers growth
   - Impressions, reach
   - Engagement rate (likes, comments, shares)
   - Link clicks, profile visits

   **Paid Advertising:**
   - Impressions, clicks, CTR
   - CPC, CPM
   - Conversions, conversion rate
   - CPA, ROAS
   - Budget spend vs. plan

   **Content / SEO:**
   - Organic traffic
   - Keyword rankings
   - Backlinks acquired
   - Content engagement

3. If KPI targets exist (from strategy doc), pull them for comparison
4. Organize data into a structured format
5. Flag any data gaps: "I notice we're missing X — can you provide that, or should we note it as a gap?"
