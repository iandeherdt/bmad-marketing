# Step 3: Insights

**Progress: Step 3 of 4** — Next: Recommendations

## STEP GOAL

Extract the top insights from the analysis — the "so what?" that turns data into understanding.

## MANDATORY RULES

- Insights are not metrics — "traffic increased 15%" is a finding; "our SEO investment is paying off as organic traffic now drives 40% of conversions" is an insight
- Limit to 3-5 top insights — too many insights = no insights
- Each insight should be actionable — if it doesn't lead to a decision, it's not worth highlighting
- Separate correlation from causation

## EXECUTION PROTOCOL

1. From the analysis in Step 2, extract the top 3-5 insights:
   - **What happened** — the finding
   - **Why it matters** — the business impact
   - **Why it happened** — the likely cause (be honest about uncertainty)
   - **What it means** — the implication for strategy
2. Structure each insight:
   > **Insight: [one-line headline]**
   > The data shows [finding]. This matters because [business impact].
   > This is likely driven by [cause]. Going forward, this suggests we should [implication].
3. Identify **cross-channel patterns:**
   - Are certain channels reinforcing each other?
   - Is there a content type that works everywhere?
   - Are certain audience segments responding better than others?
4. Flag **warning signals:**
   - Declining metrics that need attention
   - Channels approaching diminishing returns
   - Audience fatigue indicators
5. Present insights and ask: "Do these insights match what you're seeing on the ground? Any context I should factor in?"
