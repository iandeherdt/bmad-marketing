# Step 1: Pre-Launch Checklist

**Progress: Step 1 of 4** — Next: Channel Setup

## STEP GOAL

Verify that everything needed for launch is ready — assets, tracking, approvals, and infrastructure.

## MANDATORY RULES

- Do NOT proceed to launch if critical items are missing
- Tracking must be verified before anything goes live
- Every checklist item needs a clear status: Ready / Not Ready / N/A

## EXECUTION PROTOCOL

1. If campaign brief exists, build checklist from deliverables. Otherwise, ask:
   - "What campaign are we launching?"
   - "What channels and assets are involved?"
2. Run through the checklist using `templates/launch-checklist.md`:

   **Creative Assets**
   - [ ] All ad creatives in final format and correct dimensions
   - [ ] All copy reviewed and approved
   - [ ] Landing page(s) built and tested
   - [ ] Email templates built and tested
   - [ ] Social posts drafted and scheduled

   **Tracking & Analytics**
   - [ ] UTM parameters defined for all links
   - [ ] Conversion tracking pixels installed
   - [ ] Google Analytics goals / events configured
   - [ ] Attribution model decided
   - [ ] Reporting dashboard set up

   **Technical**
   - [ ] Landing page load speed acceptable
   - [ ] Mobile responsive tested
   - [ ] Forms working and connected to CRM
   - [ ] Email deliverability tested (spam check)
   - [ ] Redirect URLs working

   **Approvals**
   - [ ] Creative approved by stakeholders
   - [ ] Copy approved (legal review if needed)
   - [ ] Budget approved
   - [ ] Launch date confirmed

3. Flag any NOT READY items and ask: "What do we need to resolve before launch?"
4. Present completed checklist

## MENU

- **[P] Proceed** — All critical items ready, move to Channel Setup
- **[R] Revise** — Update checklist items
- **[S] Save & Exit** — Save progress and exit

## SUCCESS METRICS

- All critical checklist items marked Ready or resolved
- Tracking confirmed as functional
- All approvals obtained
- No blockers remaining
