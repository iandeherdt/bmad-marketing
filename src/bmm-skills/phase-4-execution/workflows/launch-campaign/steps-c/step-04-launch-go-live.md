# Step 4: Launch / Go-Live

**Progress: Step 4 of 4** — Final Step

## STEP GOAL

Execute the launch — flip everything live, verify it's working, and set up monitoring for the first 48 hours.

## MANDATORY RULES

- Final QA check before flipping live
- Have a rollback plan for every channel
- Monitor actively for the first 2-4 hours after launch
- Document everything for the performance report

## EXECUTION PROTOCOL

1. **Final QA** — Last check before launch:
   - "Have you done a final review of all staged content?"
   - "Is the team aware of the launch time?"
   - "Do you have a rollback plan if something breaks?"

2. **Go-Live Sequence** — Recommend an order:
   - Landing pages / website content → go live first
   - Email sequences → activate
   - Social posts → publish or confirm schedule
   - Paid ads → unpause campaigns
   - PR / outreach → send

3. **Launch Monitoring Plan** (first 48 hours):
   - **Hour 1-2:** Check all links work, tracking fires, no errors
   - **Hour 2-4:** Monitor initial engagement metrics, ad delivery
   - **Day 1:** First performance snapshot — impressions, clicks, conversions
   - **Day 2:** Compare to projections, identify any issues
   - Set up alerts for: budget pacing, conversion drops, error spikes

4. **Document the Launch:**
   - Launch date and time
   - Channels activated
   - Budget deployed
   - Initial performance observations
   - Issues encountered and resolutions

5. Save launch record to `docs/output/launch-record-{{campaign_name}}.md`

6. Present: "Campaign is live! Here's what to watch in the next 48 hours."
