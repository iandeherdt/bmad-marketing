# Step 3: Content Deployment

**Progress: Step 3 of 4** — Next: Go-Live

## STEP GOAL

Stage all content across channels — schedule posts, upload ad creatives, set up email sequences, and prepare everything for launch.

## MANDATORY RULES

- Double-check all links before scheduling
- Verify UTM parameters on every link
- Stage in draft/paused mode — do not publish yet
- Test emails to internal addresses first

## EXECUTION PROTOCOL

1. Create a deployment checklist by channel:

   **For each piece of content:**
   - [ ] Copy finalized and proofread
   - [ ] Creative assets uploaded in correct format
   - [ ] Links verified with correct UTMs
   - [ ] Scheduled for correct date/time
   - [ ] Set to draft/paused (not live)

2. **Email Deployment:**
   - Build each email in the sequence
   - Set up automation triggers
   - Send test emails to internal team
   - Verify personalization fields work
   - Check unsubscribe link works
   - Document the sequence using `templates/email-sequence.md`

3. **Ad Deployment:**
   - Upload all creatives
   - Write ad copy for each variant
   - Set targeting per ad group
   - Set to paused
   - Verify tracking pixels fire

4. **Social Deployment:**
   - Draft all posts in scheduling tool
   - Attach media assets
   - Add links with UTMs
   - Set publish times

5. **Content / Landing Page:**
   - Final proofread
   - Check all links
   - Test forms
   - Verify mobile experience
   - Set to draft/unpublished

6. Present deployment status: "Here's everything staged and ready. Any final changes before we go live?"

## MENU

- **[P] Proceed** — Everything staged, move to Go-Live
- **[R] Revise** — Fix deployment issues
- **[S] Save & Exit** — Save progress and exit
- **[B] Back** — Return to Channel Setup

## SUCCESS METRICS

- All content staged across all channels
- All links verified with UTMs
- Test emails sent and reviewed
- Everything in draft/paused mode ready for launch
