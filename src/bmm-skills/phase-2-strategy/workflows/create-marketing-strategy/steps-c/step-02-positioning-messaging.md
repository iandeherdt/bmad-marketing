# Step 2: Positioning & Messaging

**Progress: Step 2 of 4** — Next: Channel Strategy

## STEP GOAL

Define market positioning and build a messaging framework that gives every piece of content a clear foundation.

## MANDATORY RULES

- Positioning must be differentiated — "we're better" is not positioning
- Messaging must map to audience personas from research
- Value propositions need proof points, not just claims

## EXECUTION PROTOCOL

1. Define positioning:
   - "For [target audience], [product] is the [category] that [key differentiator] because [reason to believe]."
   - Ask the user to react and iterate until it feels right
2. Build the messaging framework using `templates/messaging-framework.md`:
   - **Positioning Statement** — The core positioning
   - **Brand Promise** — What you consistently deliver
   - **Value Propositions** (3-4):
     - Each with: headline, supporting copy, proof point
   - **Messaging by Audience** — How to adjust messaging for each persona
   - **Tone & Voice** — High-level voice attributes (detailed brand guidelines come later with Claire)
   - **Key Phrases / Taglines** — Candidates for consistent use
   - **Messaging Don'ts** — What to avoid saying
3. Present the framework and ask for feedback
4. Iterate until the user feels the messaging is strong and differentiated
