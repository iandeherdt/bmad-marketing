# Step 1: Brief Intake

**Progress: Step 1 of 3** — Next: Creative Direction

## STEP GOAL

Gather all the strategic inputs needed to write an effective creative brief — objectives, audience, messaging, and constraints.

## MANDATORY RULES

- Every brief needs a single, clear objective — not a laundry list
- The key message must be one sentence, not a paragraph
- Pull from strategy docs when available; don't reinvent positioning

## EXECUTION PROTOCOL

1. Ask the user:
   - "What's the campaign name or working title?"
   - "What's the single most important objective? (awareness, leads, conversions, retention, launch)"
   - "Who's the primary audience for this campaign?"
2. If messaging framework exists, ask which value proposition to lead with. If not:
   - "What's the one thing we want the audience to take away?"
   - "What proof do we have to back it up?"
3. Gather constraints:
   - "What's the budget for this campaign?"
   - "What's the timeline? When does this need to be live?"
   - "Any brand guidelines or constraints we need to follow?"
   - "Who needs to approve this?"
4. Summarize the brief intake:
   - **Campaign Name**
   - **Objective**
   - **Target Audience**
   - **Key Message**
   - **Proof Point**
   - **Budget**
   - **Timeline**
   - **Approvers**
5. Ask: "Does this capture the brief accurately?"
