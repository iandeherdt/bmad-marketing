# Step 3: Deliverable Specs

**Progress: Step 3 of 3** — Final Step

## STEP GOAL

Specify every asset needed for the campaign with dimensions, copy requirements, and production details.

## MANDATORY RULES

- Every deliverable needs specific dimensions or word counts
- Include CTA for every asset
- Group deliverables by channel
- Include a production timeline

## EXECUTION PROTOCOL

1. Based on the campaign objectives and channels, list all needed assets:
   - **Digital Ads** — sizes, platforms, copy lengths
   - **Social Posts** — platform, format (image/video/carousel), copy length
   - **Email** — subject line, preview text, body copy, CTA
   - **Landing Page** — sections, copy blocks, form fields
   - **Blog/Content** — word count, format, SEO keywords
   - **Video** — length, format, key scenes
   - **Print/Other** — dimensions, specs
2. For each deliverable, specify:
   - **Asset Name**
   - **Channel / Platform**
   - **Dimensions / Format**
   - **Headline** (from Step 2 direction)
   - **Body Copy** — word count or character limit
   - **CTA** — specific call to action
   - **Visual Notes** — specific image/design direction
3. Create a production timeline:
   - Copy drafts due
   - Design drafts due
   - Review/approval rounds
   - Final assets due
   - Launch date
4. Save using `templates/campaign-brief.md`
5. Optionally generate individual design briefs using `templates/design-brief.md`
